# ts-copilot

A  toolkit for javascript as well as typescript
## Usage

```shell
# npm
npm i ts-copilot
# yarn
yarn add ts-copilot
```

```ts
import { curry, isArrayBuffer, isRegExp } from 'ts-copilot'

isArrayBuffer(new ArrayBuffer(2))

```

## API documentation

Read jsdoc of files or run `npm run docs` then open folder docs
