# Toolkit

This's a toolkit has contained libraries such as dom, path and ts etc.

## ts-copilot

Workspace `./packages/eskit`

### Usage

```shell
# npm
npm i ts-copilot
# yarn
yarn add ts-copilot
```

## dom-copilot

Workspace `./packages/domkit`

### Usage

```shell
npm i path-copilot
```

## path-copilot

Workspace `./packages/pathkit`

### Usage

```shell
npm i path-copilot
```

# Usage

- install

```bash
npm i
npm run bootstrap
npm run test
npm run build
```
